6.0a7 (2019-09-15)

  • 未実装や問題のあった機能の改善
    • [MHTで保存]の「ファイル選択を省く」設定を実装（省かない場合も直接保存ダイアログを開くように）
    • 各種キーボードショートカットを実装 （コメント欄フォーカス、sageボタン、MHT保存、ブルマ女将） （注：キー変更は全拡張機能のショートカットキー管理からを推奨、従来設定値と非互換）
    • 設定ページの詳細設定フォーマットUIの例表示・初期化の実装 （タイトル、MHT保存のデフォルトのファイル名、画像を保存2のサブディレクトリ）
    • 修正: ツールバーボタンのアクション「レス送信モードで動かす」「外部板に登録」が機能しない問題
    • 修正: スレ操作パネルのサムネ表示サイズをハーフにするとサムネ固定が機能しない問題
  • Firefox・ふたばの仕様変更への対応
    • [続きを読む]や[同期]でスレの消滅時刻が更新されなくなった問題に対応
    • Firefox65以降「コメント欄内からスクロールする」設定が機能していなかった件に対応
  • All changes from previous release on github (21 commits, 25 files changed)
